# DataWarehouseAnalytics
## Project Overview
This project builds a Data Warehouse using MySQL and performs SQL based analytics including EDA and Advanced Analytics.

## Database
- Database: DataWarehouseAnalytics
- Tables: gold_dim_customers, gold_dim_products, gold_fact_sales

## Project Structure
- 01_database_setup/ - Database and table creation scripts
- 02_eda/ - Exploratory Data Analysis queries
- 03_advanced_analytics/ - Advanced Analytics queries

## Tools Used
- MySQL
- VS Code
- Git & GitHub
